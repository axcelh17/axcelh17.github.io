Resumen Profesional

Mi interés y habilidad por el desarrollo de software se consolidó gracias a una experiencia práctica: la creación de un sistema de menú digital para una cafetería, donde implementé la base de datos de Firebase para manejar la información de los productos de manera dinámica y eficiente.

Experiencia Profesional

Actualmente, estoy colaborando con una psicóloga en el desarrollo de una base de datos robusta para la gestión de pacientes, incluyendo los siguientes proyectos específicos:

  • Diseño de la Estructura (Schema): Creación del modelo relacional de la base de datos para asegurar la integridad de los datos de los pacientes (información personal, historial médico y citas).
  • Sistema de Registro y Búsqueda: Implementación de funcionalidades para el alta rápida de nuevos pacientes y un motor de búsqueda avanzado basado en distintos criterios de salud.
  • Módulo de Seguimiento de Sesiones: Desarrollo de una tabla dedicada para el registro cronológico de cada sesión, incluyendo notas, diagnóstico temporal y planes de acción (todo basado en datos).
  • Gestión de Citas Recurrentes: Configuración de la base de datos para manejar la lógica de citas futuras, recordatorios y manejo de disponibilidad de horarios del terapeuta.
  • Reportes de Progreso Basados en Datos: Creación de consultas SQL para generar reportes que miden el progreso del paciente a través del tiempo, utilizando métricas específicas de estado de salud.
